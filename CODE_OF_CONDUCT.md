# Code of Conduct

**DebtBomb — v0.3.0**

DebtBomb is an open-source project built to help teams control and eliminate technical debt.  
To make sure this project remains a healthy, welcoming, and productive space, we require all participants to follow this Code of Conduct.

---

## Our Pledge

We pledge to make participation in DebtBomb a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity.

We are committed to fostering a respectful, constructive, and collaborative community.

---

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Using welcoming and inclusive language  
- Being respectful of differing viewpoints and experiences  
- Gracefully accepting constructive criticism  
- Focusing on what is best for the project  
- Showing empathy toward other community members  

Examples of unacceptable behavior include:

- Trolling, insulting, or derogatory comments  
- Personal attacks, harassment, or threats  
- Publishing others’ private information without explicit permission  
- Sexualized language or imagery  
- Deliberate intimidation, stalking, or sustained disruption  

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the maintainers via GitHub issues, pull requests, or direct communication.

Maintainers have the right and responsibility to remove, edit, or reject contributions and to ban temporarily or permanently any contributor for behavior they deem inappropriate, threatening, offensive, or harmful to the project or its community.

---

## Scope

This Code of Conduct applies within all project spaces, including:

- GitHub issues  
- Pull requests  
- Discussions  
- Documentation  
- Community forums and social spaces related to DebtBomb  

It also applies when representing the project in public spaces.
